# pix2pix Sketch Translation Models

This repository contains the pre-trained PyTorch weights for my pix2pix sketch-to-image translation project.

## ⚠️ Where to find the model weights
Because the `.pth` files are over 200MB, they are hosted in the GitHub Releases section. 
👉 **[Download the .pth weight files here](../../releases/latest)**
